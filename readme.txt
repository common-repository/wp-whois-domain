== Installation ==
1. Upload the folder "wp-whois-domain" to "/wp-content/plugins/"
2. Activate the plugin through the "Plugins" menu in WordPress back end.
3. Go to "Whois Domain" and customize your design as per your need.
4. use below shortcode to display Whois Domain section in your page or post editor.

`
[whois-domain] 
`

4. use below shortcode to display Whois Domain section in your template file.

`
<?php  echo do_shortcode('[whois-domain]'); ?>
`
